# Sistema de Inventario

Sistema web de gestión de inventarios desarrollado con Laravel y Tailwind CSS. Permite administrar productos, controlar existencias y llevar seguimiento de movimientos dentro del inventario.

## 🚀 Tecnologías utilizadas

* PHP
* Laravel
* Tailwind CSS
* MySQL
* JavaScript

## 📦 Características

* Gestión de productos
* Control de stock
* Registro de entradas y salidas
* Panel de administración
* Interfaz responsive
